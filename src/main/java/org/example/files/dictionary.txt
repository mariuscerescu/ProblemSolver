adunare - Adunarea este o operație matematică prin care se combină două sau mai multe numere pentru a obține un total.
scădere - Scăderea este o operație matematică care implică eliminarea unui număr din altul pentru a afla diferența.
număr - Un număr este un concept de bază în matematică care indică o cantitate sau ordine.
pagină - Paginile sunt fiecare dintre fețele unei foii de hârtie dintr-o carte.
colecție - O colecție este un grup sau set de obiecte adunate împreună.
rămas - Cuvântul „rămas” reprezintă ce rămâne după ce o parte a fost luată sau eliminată.
cărți - Cărțile sunt obiecte tipărite sau scrise manual, constând din pagini legate împreună.
fotografii - Fotografiile sunt imagini create prin înregistrarea luminii sau a altor radiații pe un material sensibil.
ceasornicar - Ceasornicarul este o persoană care repară sau construiește ceasuri.
litri - Litrul este o unitate de măsură pentru volum în sistemul metric.
ecuație - O ecuație este o afirmație matematică care arată că două expresii sunt egale, folosind simbolul „=”. Notăm cu „x” necunoscuta, care va reprezenta a doua expresie, iar prima expresie va fi reprezentată de suma numerelor din problemă. Exemplu: "2 + 3 = x".
